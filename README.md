# 💊 Bilinçli Doz & Dijital Şifahane

**Akılcı İlaç Kullanımı için Yaşlı Dostu Web Sitesi**

---

## ✨ Özellikler

✅ **5 Altın Soru** - Her ilaç için sorulması gereken temel sorular  
✅ **İlaç Arama** - İlaç adına göre bilgi arama  
✅ **Yan Etkiler** - Yan etkiler ve acil durumlar  
✅ **Kullanım Rehberi** - Adım adım talimatlar  
✅ **Yaşlı Dostu** - Büyük yazılar, basit menüler, yüksek kontrast  
✅ **Mobil Uyumlu** - Telefondan açıldığında mükemmel çalışır  

---

## 🚀 HIZLI BAŞLAMA

### 1️⃣ Vercel'de Deploy Edin (Önerilen)

**Adım 1:** https://vercel.com adresine gidin  
**Adım 2:** "Sign up" → "Continue with GitHub" (veya e-mail)  
**Adım 3:** "New Project" → GitHub repo'nuzdan yükleyin  
**Adım 4:** "Deploy" → Bitti! ✅

**Aldığınız URL:** `https://bilinclidoz.vercel.app`

### 2️⃣ QR Kod Oluşturun

1. `QR_KOD.html` dosyasını açın
2. "İndir" butonuna tıklayın
3. QR kodunu kartlarınıza yazdırın

### 3️⃣ Custom Domain (İsteğe Bağlı)

`bilinclidoz.app` alan adını almak istiyorsanız:
- Namecheap'ten satın alın (~$6/yıl)
- Vercel'de Domain settings'e gidin
- Alan adını bağlayın
